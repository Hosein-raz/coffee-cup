## 📖 About The Project

A realistic, 3D-like illustration of a steaming coffee/tea cup built entirely with pure HTML and CSS. This project goes beyond basic styling to demonstrate advanced CSS art techniques, creating depth and dimension without any external images or SVGs. The highlight of the project is a highly realistic, continuous smoke animation that brings the static elements to life.

## ✨ Key Technical Features

This repository showcases several advanced frontend design techniques:

* **Pure CSS 3D Art:** The cup, handle, liquid, and plate are crafted using semantic HTML elements styled with `border-radius`, custom `linear-gradient` lighting, and complex `box-shadow: inset` combinations to create a perfect 3D illusion.
* **Realistic Smoke Animation:** Utilizes custom `@keyframes` to animate the vertical translation, horizontal scaling (`scaleX`), and opacity of the smoke particles, mimicking how real steam dissipates in the air.
* **CSS Filters:** Applies the `filter: blur(8px)` property to the smoke spans to give them a soft, vapor-like and natural appearance.
* **Staggered Delays via CSS Variables:** Employs CSS custom properties (`--i` variables) injected inline within the HTML to cleverly randomize and stagger the `animation-delay` of each individual smoke particle, ensuring a smooth, organic flow.

## 🚀 How to Use

You can easily explore or integrate this CSS art into your own projects:

1. Clone the repository:
```bash
git clone https://github.com/Hosein-raz/coffee-cup.git
```
2. Open the `index.html` file in your browser to see the animation in action.
3. To use the elements in your project, copy the `.cup` and `.smoke` containers and import the relevant styles from `style.css`.

## 📂 Project Structure

```
coffee-cup/
├── index.html    # The semantic structure and inline CSS variables for the smoke
└── style.css     # 3D styling, lighting gradients, and keyframe animations
```
